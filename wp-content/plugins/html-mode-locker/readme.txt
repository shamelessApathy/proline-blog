Adds and option to lock post editor in HTML Mode on selected post types on per-item basis.

== Description ==
We all experienced frustration of using WYSIWYG editor especially when switching between Visual and HTML modes. If you have a need to use Visual editor in your day-to-day operations, but you also need the ability to have some posts or pages in pure HTML format - HTML Mode Locker allows you to do that on per-post basis.

== Installation ==

1.	Install and activate like any other basic plugin.

2.	Go to Settings => Writing and scroll down until you see HTML Mode Locker section.

3.	Activate HTML Mode Locker on desired post types and save your settings.

The HTML Locker meta box will appear to the right of the content editor on selected post types.
